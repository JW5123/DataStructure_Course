#include <bits/stdc++.h>
using namespace std;
函式main{
    宣告一整數二維陣列adj用以建圖
    宣告一整數陣列indeg紀錄目前該節點被多少邊連到
    宣告m, n為整數型態變數
    輸入節點數量和邊的數量
    for迴圈輸入n次{
        宣告from, to為整數型態變數
        輸入節點兩端點的編號
        將兩端點的編號存入adj中建圖
        紀錄圖上各點被幾條邊連到
    }
    宣告topo,head,tail儲存只有出邊無入邊的點和佇列頭尾位置
    for迴圈輸入m次{
        如果沒有被任何邊連向的點則放入topo中
    }
    當佇列中還有元素時執行while迴圈直到無元素為止{
        每次從佇列前端取出v,並刪除由v點連向每個點u的邊
        若刪除後為0則放入topo中
    }
    如果tail小於m表示出現環則輸出No a DAG
    否則依序輸出topo中的元素
}
