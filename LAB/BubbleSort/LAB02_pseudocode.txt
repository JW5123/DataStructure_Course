宣告函式BubbleSort(宣告參數arr[], n為整數) {
    初始值為0並向後搜尋n次(for){
        初始值為0並向後逐一比較陣列相鄰兩數(for){
            當前一個數值大於後面數值{
                將兩數進行交換;
            }
        }
    }
}
