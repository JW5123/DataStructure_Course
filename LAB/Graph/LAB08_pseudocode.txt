#include <bits/stdc++.h>
using namespace std;
定義一個Ｎ大小為50000
宣告一個陣列adj儲存與他相鄰的節點
宣告一個陣列w儲存節點價值
宣告一個陣列Visit儲存走過的節點
宣告DFS函式{
    宣告整數weight儲存走訪過的價值
    將Visit設為走訪過
    for迴圈走訪與該節點有關的相鄰節點{
        檢查該節點是否走訪過
        未走訪過則將當前走訪過的價值總和加上當前節點價值
    }
    走訪過則回傳當前價值總和
}
宣告main函式{
    宣告m, n, MAX, total為整數型態並將MAX及total初始化為0
    輸入寶藏地點和道路的數量
    for迴圈輸入m次{
        輸入寶藏的價值
        計算價值總和並存到total
    }
    for迴圈輸入n次{
        宣告a, b為整數型態
        輸入道路兩端點的編號
        將道路兩端點的編號存入adj中建圖
    }
    for迴圈走訪所有寶藏地點{
        判斷是否已走訪過
        比較當前節點是否為最大如果是則賦值給MAX
    }
    輸出最大價值
    輸出最大價值和所有寶藏價值和
}
